## Install

```bash
npm install is-in-editor
```

```bash
yarn add is-in-editor
```

```bash
pnpm add is-in-editor
```

## Usage

```ts
import { isInEditor } from 'is-in-editor'

isInEditor()
// => true if running in an editor
```

## Supported Editors and IDEs

- VSCode
- JetBrains IDE
